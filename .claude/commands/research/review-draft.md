# Review Draft

Review draft text for quality and clarity.

## Usage
```
/research/review-draft [section or file]
```

## Examples
```
/research/review-draft abstract
/research/review-draft paper_body.md
/research/review-draft "lines 50-100"
```

## Invokes
- **draft-reviewer** agent

## Review Criteria
- Clarity of argument
- Precision of claims
- Logical flow
- Citation support
- Consistent terminology
- Page limit compliance

## Output Format
```
## Review Summary
[Overall assessment]

## Major Issues
1. [Issue with suggestion]

## Minor Issues
- [Small fixes]

## Strengths
- [What's working]
```
