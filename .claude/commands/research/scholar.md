# Scholar Command

Master orchestrator for research tasks.

## Usage
```
/research/scholar [task description]
```

## Examples
```
/research/scholar Write the abstract
/research/scholar Find papers on plasticity loss
/research/scholar Review the methods section
```

## Invokes
- **scholar** agent as orchestrator

## What It Does
1. Parses your research task
2. Creates an execution plan
3. Delegates to specialist agents
4. Verifies results
5. Updates memory bank

## Supported Tasks
- Writing sections
- Literature search
- Citation verification
- Draft review
- LaTeX building
- Progress tracking
