# Rewrite

Rewrite text in direct, academic style.

## Usage
```
/util/rewrite [text or selection]
```

## Style Guidelines
- Direct, active voice
- PhD-level precision
- No fluff or filler
- Concrete over abstract
- One idea per sentence

## Transformations
| Before | After |
|--------|-------|
| "It can be seen that" | [delete] |
| "In order to" | "To" |
| "A number of" | "[specific number]" |
| "Due to the fact that" | "Because" |
| "was performed" | "[subject] did" |

## Example
**Before**: "It is important to note that the results which were obtained from the experiments that were conducted demonstrate significant improvements."

**After**: "Our experiments show a 15% improvement in accuracy."
